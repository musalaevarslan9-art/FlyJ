# Lucky Jet Crash — Static Demo (GitHub Pages)

Это **статический** (без сервера) демо‑сайт crash game:
- всё работает **в браузере** (HTML/CSS/JS)
- баланс демо: **$10,000**
- ставки, кэшаут, авто‑кэшаут, история коэффициентов, звук, анимации
- сохранение баланса/истории в `localStorage`

## Запуск локально
Просто открой `index.html` двойным кликом (лучше через Live Server, но не обязательно).

## Публикация на GitHub Pages за ~30 секунд
1. Создай репозиторий на GitHub (например `luckyjet-demo`)
2. Залей содержимое папки в корень репозитория:
   - `index.html`
   - `styles.css`
   - `app.js`
   - папка `assets/`
3. GitHub → **Settings → Pages**
4. **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: `main` / `(root)`
5. Сохрани — через минуту сайт будет доступен.

> ВАЖНО: это демо без реальных денег/депозитов/выводов, без backend.
